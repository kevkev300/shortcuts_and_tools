Mac


shortcut description
cmd + tab change programm to the right (order: currently used)
cmd + shift + tab change programm to the left (order: currently used)
ctrl + right-arrow change window to the right
ctrl + left-arrow change window to the left
cmd + q quit current programm
cmd + w close current window
cmd + ctrl + q lock screen
cmd + shift + 4 screenshot with selector
cmd + , open options/preferences
cmd + space open spot light search
cmd + m minimize current window

Atom

I use the platformio-ide-terminal plugin.

shortcut description
cmd + opt + right-arrow next tab to tde right
cmd + opt + left-arrow next tab to tde left
cmd + p open file
cmd + f search in file
cmd + shift + f search in project
cmd + opt + f focus terminal (and back to editor)
cmd + opt + t open new terminal
ctrl + 0 toggle tree & editor view

VS Code

shortcut description
cmd + q cmd + q quit programm
cmd + ctrl + right drag file to right split view (and open that view if non-existent)
cmd + opt + left-arrow next tab to the left and copy current file
cmd + p open file
cmd + f search in file
cmd + shift + f search in project
cmd + shift + p command line
ctrl + cmd + up/down resize terminal
cmd + shift + p + "rename terminal" rename terminal
cmd + opt + f focus terminal
cmd + opt + t open new terminal
cmd + 0 focus directory view
cmd + 1 focus code view

Browser (mainly Chrome)


shortcut description
control + tab / cmd + opt + right-arrow change tab to the right
control + shift + tab / cmd + opt + left-arrow change tab to the left
cmd + left-arrow go back
cmd + right-arrow go forward
cmd + t open new tab
cmd + w close current tab
cmd + q (twice) quit chrome
cmd + l put cursor into url bar
